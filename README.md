## How to play
🎮 To play the game, download the latest version from the [Releases](https://github.com/Luk-gitgood/Game-PO5/releases) page, unzip and run the executable! 

## Instructions:
- Use f11 for fullscreen

- Esc for settings in game

- Use WASD controls for moving

- Use LMB for attacking

- Obtain new weapons by killing more enemies!
## Weapons
Unlock weapons by killing enemies:

- 1 = dagger
- 2 = revolver
- 3 = shotgun
- 4 = sniper

## Stack used for project
- Tiled for map creation
- Pygame library
- Audacity & Ableton for music / sfx
